# Minesweeper game

Create with:

-   React (NextJs)
-   TypeScript
-   CSS modules

Dependencies:

-   zustand
-   sweetalert2
-   react-confetti

## Known issues

-   when active: mine marker & loupe. If loupe is active and double click on a number reveals all the left cells, then the game does not show the 'wonModal' as expected

```
npm run dev
```
