# Minesweeper game

Create with:

-   React (NextJs)
-   TypeScript
-   CSS modules

Dependencies:

-   zustand
-   sweetalert2
-   react-confetti

## Known issues

-   if wrong cell is marked as mine when revealing nulls, then mines to discover counter will have missing mines

## How to run

```
npm run dev
```

## StoryBook

```
npm run storybook
```
